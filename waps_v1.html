Processing Steps

  • Use WhatsApp on mobile phone
  • Select the chat group you want to analyze
  • Visit group information, scroll down to the end
  • Select "Export Chat"; Do not include media
  • Save generated chat file (usually a zip file)
  • Extract the main text file from ZIP file
  • Load the file here for analysis
  • Adjust the session duration as desired

Note on security

  • All processing is done on your machine
  • No chat information is sent over the internet
